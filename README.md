## Installation

Clone the repository:

```sh
git clone ssh://git@github.com/CodeEditorLand/Sublet.git --recurse-submodules --
```

Using NPM:

```sh
npx astro add @playform/compress
```

Build the Editor:

```sh
cargo tauri build
```
